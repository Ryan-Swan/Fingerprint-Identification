User guide - Bookings

Make booking

On the admin dashboard, click the create button in the bookings session. This will redirect you to the bookings page

Fill in the form and submit. It will redirect you to the admin page if successful.

Modify / Delete booking

On the admin dashboard, within the bookings section, click the booking you want to change

For example, in the example above, click the blue '00:00 to 23:00' button

Fill in the form and submit

Alternatively, you can use list view for bookings
